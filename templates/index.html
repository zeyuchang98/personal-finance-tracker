Welcome! A quick setup first

The app needs two API keys to work. Add the missing one(s) to the .env file in the project folder (copy .env.example to .env if it doesn't exist yet):

    1. Open .env in any text editor and paste the key(s) in.
    2. Make sure PLAID_ENV matches your Plaid secret (production for real banks, sandbox for test data).
    3. Click Recheck — no restart needed.
